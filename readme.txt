## LabsIncor

Inspirado pelo LabHCmed, criei um progrma para extrair os exames do InCor em formato de texto, para simplificar na hora de canetar os doentes. Após Baixar o .exe, basta executa-lo. É necessário baixar o PDF do exame por enquanto. Após isso, selecione o arquivo e clique em "Converter a Texto". Copiar copia o conteudo da tela. Limpar limpa a Tela.
